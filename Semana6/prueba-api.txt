Al enviar una petición POST a http://localhost:3000/registro con el JSON:
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor responde con:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto significa que el servidor recibió correctamente los datos enviados
en el cuerpo de la petición gracias a req.body y los devolvió en formato JSON.
