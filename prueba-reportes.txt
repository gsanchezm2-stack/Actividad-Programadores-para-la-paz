Al enviar una petición POST a http://localhost:3000/reportes con un JSON,
el servidor registra el reporte en la lista y responde confirmando que fue guardado.

Al consultar GET http://localhost:3000/reportes, el servidor devuelve la lista
completa de reportes registrados en memoria. Por ejemplo:

[
  {
    "id": 1,
    "tipo": "Infraestructura",
    "descripcion": "Daño en alumbrado público"
  },
  {
    "id": 2,
    "tipo": "Ambiente",
    "descripcion": "Acumulación de residuos en una zona comunitaria"
  }
]

Cada reporte tiene un id único que se asigna automáticamente.
